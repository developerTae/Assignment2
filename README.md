파일 비교 프로그램 구현
  • linux ubuntu(18.04 LTS)상에서 두 개의 파일을 비교하는 프로그램 구현
  • 다음과 같은 기능을 구현하되, milestone과 issue를 이용해서개발을 진행
      1. 파일 정보 받아 오기
      2. 파일 시간 받아 오기
      3. 파일 사이즈 비교
      4. 파일 블락 수 비교
      5. 파일 수정 날짜(월/일) 비교
      6. 파일 수정 시간(시/분) 비교

위의 과정을 진행하면서 git을 이용한 팀프로젝트를 진행한다. //
